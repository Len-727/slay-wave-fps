# SLAY WAVE

C++ と DirectX 11 でフルスクラッチ開発した、ゴシック調ウェーブ制FPSです。

暗く重厚な世界で押し寄せる敵の波を撃退し、盾を投げつけてリアルタイムに敵を切断しながら生き残れ。

---

## 特徴

### 戦闘システム
- **盾投げ** — 盾を敵に投げつけると、リアルタイム**メッシュスライス**で動的に切断される
- **チャージ突進** — 盾を構えてエネルギーを溜め、強力な突進攻撃を繰り出す
- **射撃** — 銃で敵を撃ち、リロードしながら激化する波を生き延びる

### 描画
- **HLSL** によるマルチパスレンダリングパイプライン
- **コンピュートシェーダー** によるGPUパーティクルシステム（血液、火炎、薬莢）
- ボーン駆動のスケルタルアニメーション
- **Effekseer** によるリボントレイル・VFX

### 物理演算
- **Bullet Physics** による剛体物理シミュレーション
- メッシュスライス時の破片・欠片の飛散

### 敵AI
- ナビゲーショングリッド上の**A*経路探索**
- 有限状態マシンによる巡回・追跡・攻撃の行動制御
- インスタンスドレンダリングによる大量描画

### 設計
- 責務ごとにファイルを分離（Physics、Debug、Render など）
- `constexpr` による定数管理
- **ImGui** デバッグオーバーレイによるリアルタイム検査

## 操作方法

| キー | アクション |
|------|-----------|
| W A S D | 移動 |
| マウス | 視点操作 |
| 左クリック | 射撃 |
| 右クリック（長押し） | 盾を構える／チャージ |
| 右クリック長押し + 左クリック | チャージ突進 |
| E | 盾投げ |
| Shift | ダッシュ |
| R | リロード |

## ビルド方法

### 必要環境
- **Visual Studio 2022**
- Windows 10 / 11



### 手順
1. このリポジトリをクローンする
2. Visual Studio 2022 で `Rivalsgame1.sln` を開く
3. **Release**（x64）でビルド・実行する

## 技術概要

| 項目 | 詳細 |
|------|------|
| 言語 | C++14 |
| グラフィックスAPI | DirectX 11 |
| シェーダー | HLSL（`D3DCompileFromFile` によるランタイムコンパイル） |
| 物理エンジン | Bullet Physics 3.x |
| AI | A*経路探索 ＋ 有限状態マシン |
| メッシュスライス | UV・法線補間によるリアルタイム切断 |
| パーティクル | コンピュートシェーダーによるGPUパーティクル |
